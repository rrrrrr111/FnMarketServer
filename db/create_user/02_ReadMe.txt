02_create_user.sql - скрипт создания нового пользователя БД

1) Перед вызовом скрипта нужно убедиться, что переменная среда ORACLE_HOME существует. Если её нет то необходимо создать и задать для неё путь к корневой папке клиента ORACLE.

Пример: set ORACLE_HOME=C:\oracle\product\10.2.0\client_1

2) Скрипт вызывается из sqlplus под пользователем SYS(sys/pass@TNS as sysdba) или под пользователем, который имеет права администратора БД(username/pass@TNS) и принимает на вход 4 параметра:
 2.1 имя создаваемого пользователя
 2.2 пароль создаваемого пользователя
 2.3 Название табличного пространства(TABLESPACE) по умолчанию для создаваемого пользователя. Пространство уже должно существовать
 2.4 Название временного табличного пространства(TABLESPACE) для создаваемого пользователя. Пространство уже должно существовать.

Пример:

 call %ORACLE_HOME%\bin\sqlplus.exe "sys/pass@orcl as sysdba"  @create_user.sql username password users temp
 call %ORACLE_HOME%\bin\sqlplus.exe username/pass@orcl @create_user.sql username password users temp


При добавлении привилегий, необходимых для работы приложения, они добавляются в скрипт 02.2_grant_user.sql
В случае, если необходимо создать двух пользователей - одного для онлайна, другого - для офлайна, тогда нужно вызвать скрипт create_user.sql два раза, указав соответствующие наименования и т.п.
При работе скрипта образуются следующие журналы в той же папке, где и сам скрипт:
1. 02.1_create_user.log - протокол создания пользователя
2. 02.2_grant_user.log - протокол наделения пользователя полномочиями.
